# WeTongbu 0.9.4

Community Plugins submission release.

- Keeps all synced Markdown, images, and attachments in the current Vault by default.
- Adds explicit browser-based account authorization for Pro without changing the Free six-digit binding-code flow.
- Shows the signed-in account and plan in plugin settings.
- Lets active Pro users choose local image downloads or stable hosted image links, with a local migration option.
- Retains task-package hash, size, and safe-path validation; temporary task objects are cleaned up after completion.

Release assets: `main.js` and `manifest.json`.
